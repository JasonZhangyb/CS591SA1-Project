# CS591SA1-Project Overview
 This project uses [*Sight Engine*](https://sightengine.com/) api to recongnize a celebrity from an image ( input as an image url ) and then call the 
 [*The Movie Database*](https://www.themoviedb.org/?language=en) to get a list of related movies based on the celebrity. Data
 will automatically be stored in a local database.
 
 User can login using Twitter.
 
 ## Folders
 NodeJS: Back-end<br> 
 Angular-MovieSearch: Front-end
